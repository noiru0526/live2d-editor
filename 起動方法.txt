起動方法
======================================

【WSL2 から起動する場合】

1. Windowsターミナルを開いて WSL2 (Debian) に入る

2. 以下のコマンドを実行:

   cd /home/sose0/claude-data/app/live2d-app
   npm install
   npm start

※ WSL2 では Windows 11 の WSLg により GUI が表示されます。


【Windows から直接起動する場合 ★おすすめ★】

既にビルド済みの EXE があります:

   C:\Users\sose0\claude-data\app\live2d-app\dist\win-unpacked\Live2D Editor.exe

このファイルをダブルクリックするだけで起動します！
(インストール不要・ポータブル)

フォルダごとデスクトップなどにコピーして使えます。


【Node.js から直接起動する場合】

1. Node.js for Windows をインストール:
   https://nodejs.org/ → LTS 版をダウンロード

2. Windows ターミナル (PowerShell) で:

   cd C:\Users\sose0\claude-data\app\live2d-app
   npm install
   npm start


======================================
操作方法（基本）
======================================

1. 起動後、メニュー「ファイル > 画像を開く」または Ctrl+O で
   アバター画像 (PNG/JPG) を開く

2. 右パネル「⚡ 自動プリセット」で基本変形をセットアップ:
   - 「頭 X/Y/Z（自動）」→ 顔追跡に合わせた頭の動きを設定
   - 「まばたきセットアップ」→ 目の位置を確認して適用
   - 「口 セットアップ」→ 口の位置を確認して適用

3. プレビューボタン (P キー) でアニメーションを確認

4. 微調整したい場合:
   - 左パネルのパラメータスライダーで対象を選択
   - ⏺ REC ボタン (R キー) を ON
   - 頂点をドラッグして変形
   - スライダーを中立に戻す

5. メニュー「ファイル > アバターHTMLを出力」で
   OBS に読み込めるアバターファイルを生成


======================================
キーボードショートカット
======================================

Ctrl+O  : 画像を開く
Ctrl+S  : モデルを保存
Ctrl+Z  : 元に戻す
Ctrl+E  : HTML 出力

E       : 編集モード
P       : プレビューモード
R       : REC オン/オフ
M       : メッシュ表示切替
X       : ミラー編集切替
O       : オニオンスキン切替
F       : 表示リセット (フィット)
B       : ブラシ選択
S       : スカルプト変形
L       : 投げ縄選択
G       : グリッドスナップ

1-9     : パラメータ 1〜9 を選択
スクロール : ズームイン/アウト
中クリックドラッグ : パン
右クリック : コンテキストメニュー
F12     : 開発者ツール
